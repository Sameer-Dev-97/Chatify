# Chatify 🗨️

A real-time chat application built with the MERN stack (MongoDB, Express.js, React, Node.js) and Socket.IO. Chatify allows users to sign up, log in, and engage in instant messaging with other users in one-on-one or group conversations.

## 🚀 Features

- 🔐 Authentication (JWT-based)
- 💬 Real-time messaging using Socket.IO
- 👥 One-on-one & Group Chats
- 📥 Message history persistence (MongoDB)
- 📱 Responsive design
- 📷 Profile pictures
- ✅ Online/offline user indicators
- 🌓 Dark/light theme toggle *(optional)*

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Redux Toolkit
- Axios
- Socket.IO-client
- Tailwind CSS / Bootstrap / CSS Modules *(pick based on your implementation)*

**Backend:**
- Node.js
- Express.js
- MongoDB (with Mongoose)
- JWT for authentication
- Socket.IO
